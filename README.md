# Philip Ajayi - Data Science & Analysis Portfolio

This is a simple one-page portfolio website built for GitHub Pages.

## Files
- `index.html` - main portfolio page

## How to publish on GitHub Pages
1. Create a new public repository on GitHub. Example: `philip-portfolio`
2. Upload `index.html` into the repository
3. Go to **Settings** -> **Pages**
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` and `/root`
5. Save
6. GitHub will give you a live link
